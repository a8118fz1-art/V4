📡 ANT+ 踏頻器連接說明
ANT+ 無法直接透過瀏覽器存取，需要橋接方案
方案 A — USB ANT+ Dongle（推薦）
1. 插入 ANT+ USB Dongle（Garmin / Wahoo 通用）
2. 安裝 ANT+ Bridge 或 GoldenCheetah 橋接軟體
3. 橋接程式將 ANT+ 數據轉為 WebSocket
4. APP 透過 ws://localhost:1234 接收踏頻數值
方案 B — 雙模踏頻器（最簡單）
選購同時支援 ANT+ + BLE 的踏頻器
例：Garmin Cadence Sensor 2、Wahoo RPM
→ 直接用 BLE 連接，ANT+ 留給碼表使用
→ 點上方「+ BLE 踏頻器」即可連接
方案 C — WebSocket 橋接（進階）
若已有 ANT+ Dongle，可使用下方測試 WS 連線
